# Instituto EDCSST - Instrucciones para Agentes Copilot

## Contexto del Proyecto
- **Framework**: Laravel 13.8
- **PHP**: 8.3+
- **Frontend**: Tailwind CSS, Alpine.js, Vite
- **Stack**: Laravel Breeze, Spatie Permissions, Excel (Maatwebsite), PDF (DomPDF)
- **Testing**: PHPUnit 12.5
- **Base de datos**: Migraciones Laravel

## Convenciones de Código

### PHP/Laravel
- PSR-4 autoload: `App\` → `app/`
- Controladores en `app/Http/Controllers`
- Modelos en `app/Models`
- Traits de negocio en `app/Traits`
- Requests en `app/Http/Requests`
- Services en `app/Services`
- Migraciones en `database/migrations`

### Frontend
- Componentes Blade en `resources/views/components`
- Layouts en `resources/views/layouts`
- CSS en `resources/css`
- JavaScript en `resources/js`

## Reglas de Implementación

### 1. Modelos y Migraciones
- Crear migraciones antes de modelos
- Usar timestamps (`created_at`, `updated_at`)
- Definir fillable/guarded explícitamente
- Incluir relaciones tipadas (retorno con tipos)

### 2. Permisos
- Usar Spatie Permissions para roles/permisos
- Verificar permisos en middleware y modelos
- Definir permisos en seeders

### 3. API/Rutas
- Rutas en `routes/web.php` o `routes/api.php`
- Usar resource controllers cuando sea apropiado
- Validación en Form Requests (`app/Http/Requests`)

### 4. Vistas Blade
- Usar components con props tipados
- Directivas: `@auth`, `@can('action')`, `@forelse`
- Classes Tailwind: responsive-first, usar prefijos (`md:`, `lg:`)

### 5. Testing
- Tests en `tests/`
- Usar factories con Faker
- Archivo config: `phpunit.xml`
- Ejecutar: `composer test`

## Tareas Comunes

### Setup Inicial
```bash
composer setup
npm run dev  # En otra terminal
```

### Migraciones
```bash
php artisan make:migration create_table_name --create=table_name
php artisan migrate
php artisan migrate:rollback
```

### Modelos
```bash
php artisan make:model ModelName -m  # Con migracion
php artisan make:request CreateModelRequest
```

### Testing
```bash
composer test
# O específico:
php artisan test tests/Feature/ModelTest.php
```

## Archivos Críticos
- `.env` - Variables de entorno
- `config/` - Configuración global
- `routes/` - Definición de rutas
- `database/` - Migraciones y seeders
- `app/` - Lógica de negocio

## Dependencias Principales
- `anhskohbo/no-captcha` - reCAPTCHA
- `barryvdh/laravel-dompdf` - Generación PDF
- `maatwebsite/excel` - Exportar/importar Excel
- `spatie/laravel-permission` - Gestión de permisos

## Comandos Útiles
```bash
php artisan serve              # Servidor local
php artisan tinker             # Console interactiva
php artisan migrate            # Ejecutar migraciones
php artisan seed               # Ejecutar seeders
php artisan optimize:clear     # Limpiar caché
npm run dev                    # Dev Vite
npm run build                  # Build producción
```
